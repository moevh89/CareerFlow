## 2026-07-13 - Duplicated Request Parsing\n\n**Learning:** Found repetitive `json_decode(file_get_contents('php://input'), true)` and CSRF validation logic across multiple controllers. This was partly due to `AuthController` not extending the base `Controller`, leading to duplicated utility methods like `jsonResponse`.\n\n**Action:** Moved the parsing and validation logic into base `Controller` methods (`getJson()` and `validateCsrf()`). Ensure all new controllers extend the base `Controller` to reuse these utilities and avoid duplicating request handling logic.
